About

- history of periodic table -

My name's Victor Temprano, and I wanted to make something a bit more visual to help while learning about chemistry. Bohr diagrams are great, but 3D models are fun to play around with, and can really help us visualize what happens at a small scale (though very imperfectly).

Phase 1 and 2 of this project include:

  • Intro animation of some kind, or telling user what to do
  • Cool 3D information about element appearing near it
  • Selectable/checkbox for electrons, protons, neutrons
  • Easy zoom and move around buttons
  • Highlighting/checkbox for active orbitals and sub-orbitals
  • Adding spectrum information and more for each element
  • Mobile design and sliding periodic table
  • Searchable periodic table list
  • Some cool links/images for the elements
  • Ability to create and move elements around into custom configurations
  • Elements have charge and can stick together
  • Relevant equations or scientific names for orbitals
  • Each element have a zoom level

See the Github page for more information, or check out my page to get in touch.
